Installation
------------

  1. Install Prezto:

	https://github.com/sorin-ionescu/prezto

  2. Clone the dotfiles repository:

	git clone --recursive https://github.com/icereval/dotfiles.git "$HOME/.dotfiles"

  3. Link custom dotfile configurations:

	$HOME/.dotfiles/link.zsh
